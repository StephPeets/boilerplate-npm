Manage Node.js projects and npm packages using package.json

Part 1 of Free Code Camp Backend Challenges

Semantic Versioning: "package": "MAJOR.MINOR.PATCH"
ex. "mypackage": "1.0.6"

  • Add a tilde symbol (~) as prefix to allow npm to update to the latest patch release.

  • Add a caret symbol (^) prefix to allow npm to update to latest minor release.
